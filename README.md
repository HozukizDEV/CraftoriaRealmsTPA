Sobre o Plugin

Este plugin foi desenvolvido exclusivamente para o servidor CraftoriaRealms com a finalidade de implementar um sistema de teleporte personalizado, permitindo que os jogadores solicitem e aceitem teleportes de forma interativa e dinâmica.

A ideia do plugin é proporcionar uma maneira simples e divertida de os jogadores se teletransportarem entre si, com uma interface intuitiva e funcionalidades que se integram ao sistema já existente do servidor.

Compatível com Minecraft 1.8.8, o plugin garante estabilidade e flexibilidade, permitindo futuras atualizações e integrações no servidor.

Funcionalidades Principais

Sistema de Solicitação de Teleporte: Jogadores podem enviar pedidos de teleporte para outros jogadores e aguardar aprovação.

Comandos Customizados:

/call <jogador> - Envia um pedido de teleporte para o jogador especificado.

/bring <jogador> - Aceita o pedido de teleporte de outro jogador e realiza o teleporte.

Cooldown de 30 segundos: Impede que os jogadores utilizem os comandos em excesso, garantindo equilíbrio no uso.

Mensagens Interativas: Quando um pedido de teleporte é enviado, o destinatário recebe uma mensagem interativa com a opção de aceitar ou rejeitar o teleporte.

Integração com Eventos do Servidor: Total compatibilidade com os sistemas existentes no servidor, como gerenciamento de permissões e eventos de jogador.

Pré-configuração: O plugin já vem configurado para o uso imediato, sendo possível ajustar traduções e mensagens através de arquivos .yml.

Requisitos

Servidor Minecraft versão 1.8.8.

Java 8 ou superior instalado no servidor.

Permissão para adicionar plugins na pasta /plugins.

Comandos Disponíveis

/call <jogador>
Envia um pedido de teleporte para o jogador especificado. O destinatário recebe uma mensagem interativa para aceitar ou rejeitar o pedido.

/bring <jogador>
Aceita um pedido de teleporte de um jogador e realiza o teleporte para o local onde o jogador solicitante está.

Instalação

Faça o download do plugin (arquivo .jar).

Coloque o arquivo dentro da pasta /plugins do seu servidor.

Reinicie o servidor.

Verifique o console para garantir que o plugin foi carregado corretamente.

Uso

Para enviar um pedido de teleporte:
O jogador utiliza o comando /call <jogador>. O jogador alvo receberá uma mensagem interativa com a opção de aceitar ou usar o comando /bring para aceitar o teleporte.

Para aceitar um pedido de teleporte:
O jogador alvo utiliza o comando /bring <solicitante>. O teleporte será realizado, e ambos os jogadores receberão uma mensagem confirmando o teleporte.

Cooldown:
O sistema impõe um cooldown de 30 segundos entre os usos dos comandos, evitando uso excessivo e mantendo o equilíbrio do jogo.

Exemplo de Uso

Jogador A executa /call Jogador B.

Jogador B recebe uma mensagem interativa perguntando se ele aceita o teleporte.

Jogador B clica em "Clique Aqui" ou usa o comando /bring Jogador A para aceitar.

Jogador A é teletransportado para Jogador B (ou vice-versa, dependendo da configuração do teleporte).
